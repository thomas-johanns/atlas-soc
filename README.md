# Atlas SoC

A custom System on Chip built in SystemVerilog featuring:

5 stage pipelined RISC-V processor
Memory mapped peripherals
UART
Timer
VGA graphics controller
Simple game demo 

## Goals

Learn processor design
Learn system integration
Explore ASIC design flows
Build a complete hardware/software project

## Current Status

Planning Phase

## Planned Architecture

CPU
├── Instruction Memory
├── Data Memory
├── Timer
├── UART
└── Graphics Controller
